<strong>Beschreibung:</strong>
Addon zum editieren und anlegen von CSS Dateien.

Das sollte zum jetzigen Zeitpunk möglich sein:
<ul>
	<li>Es werden alle CSS-Dateien, die im Medienpool hinterlegt sind, zum bearbeiten angeboten.</li>
	<li>Es sollten momentan nur CSS Dateien angelegt werden können (Bspl.: dateiname.css). TXT kommt später.</li>
	<li>Das Editorfenster sowie die Vorschau sollten genauso dargestellt werden, wie die CSS-Datei selbst.</li>
	<li>Die Höhe des Editorfensters kann über Setup verändert werden.</li>
	<li>Das löschen einzelner CSS-Dateien vom Server und aus dem Medienpool.</li>
	<li>Es sollte ein Hinweis erscheinen, bei Eingabe von falschen Eigenschaften.
		background <- richtig, keinen Hinweis
		backgroond <- Ausgabe eines Hinweises
	</li></ul>
