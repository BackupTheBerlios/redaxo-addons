<strong>Beschreibung:</strong>

Das History Addon dient dazu dem Benutzer der das Recht "history[]" hat, einen Überblick zu geben, welche Artikel/Templates/Module/Aktionen er bearbeitet hat.

Somit kann er schneller wieder in die zuletzt bearbeiteten Daten hineinspringen.


<strong>Download:</strong>

<a href="http://www.redaxo.de/18-0-addons.html">REDAXO Addon-Sammlung</a>


<strong>Installation:</strong>

- Unter "redaxo/include/addons" einen Ordner "history" anlegen

- Alle Dateien des Archivs nach "redaxo/include/addons/history" entpacken

- Im Redaxo AddOn Manager das Plugin installieren

- Im Redaxo AddOn Manager das Plugin aktivieren

- Dem Benutzer das recht "history[]" verleihen

- fertig ;)
