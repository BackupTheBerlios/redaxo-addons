<strong>Beschreibung:</strong>

Framework zum leichteren Erstellen von Addons.

Bietet Klasse zur Auflistung von Einträgen (rexList) und Bearbeitung dieser (rexForm).


<strong>Download:</strong>

<a href="http://www.redaxo.de/18-0-addons.html">REDAXO Addon-Sammlung</a>


<strong>Installation:</strong>

- Unter "redaxo/include/addons" einen Ordner "addon_framework" anlegen

- Alle Dateien des Archivs nach "redaxo/include/addons/addon_framework" entpacken

- Im Redaxo AddOn Manager das Plugin installieren

- Im Redaxo AddOn Manager das Plugin aktivieren

- Dem Benutzer das recht "addon_framework[]" verleihen

- fertig ;)
